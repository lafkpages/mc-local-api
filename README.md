# MC Local API

A Minecraft mod that provides a local HTTP REST API for interacting with and querying your Minecraft client.

GraphQL support was attempted in the past, but was not seen as a viable option and is now no longer supported.

## Use cases

This mod could be used for a variety of things. Here are some projects that use it:

- [ChunkBase Tracker](https://github.com/lafkpages/chunkbase-tracker): A user script to update the ChunkBase seed map automatically to track your in-game location.
